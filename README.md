# BigCounter

Full Screen Counter

Show fullscreen counter on the html web page.

## Usage

### KEYBOARD & MOUSE

| What | keyboard & Mouse  |
|---|---|
| Increment | Any key or click |
| Decrement | Minus sign or backspace |
| Reset zero | 0 or ESC |
| Toggle full screen | s, S or double click |
| Change background color | b, B |
| Change font | f, F |
| Help | h, H |
